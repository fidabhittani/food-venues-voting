### How to run it?

In Order to run it, Please do the following

```sh
$ Create an .env onn the root with content as below

    REACT_APP_CLIENT_ID=[FourSquare API Client ID]
    REACT_APP_CLIENT_SECRET=[FourSquare API Client Secret]

$ yarn
$ yarn start

```
